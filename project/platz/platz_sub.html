UI RENEWAL PROJECT

01. overview.

플라츠는 복합 문화 공간으로 전시나 프로젝트 정보를 알려주고 있습니다.

기존 웹사이트는 주로 이미지와 글로 구성된 정적인 웹 사이트였습니다.

사람들의 흥미를 이끌도록 화려한 디자인으로 제작하고자 했습니다. 또한 javascript와 jQuery를 이용하여 동적인 화면을 구현시켜 정보를 전달하는 것을 목표로 하였습니다.

03. Publishing.

04. Review.

  • # 처음 시도한 가로스크롤

    position:sticky와 javascript를 이용하여 특정 위치에서 가로 스크롤를 적용 시켜보았습니다.
    다만 스크롤이 끝나고 다음 화면으로 넘어갈 때 매끄럽게 적용시키지 못한게 아쉬웠습니다.

  • # 버튼이 눌리지 않음

    슬라이드를 구현할때 버튼이 눌리지 않고 cusor:pointer; 도 적용되지 않아 당황스러웠습니다.
    버튼을 먼저 작성하여 우선순위가 밀리게 되었고, z-index 값으로 우선 순위를 변경하여 해결했습니다.

  • # 자동 슬라이드

    jQuery의 setInterval()을 이용하여 자동 슬라이드를 이미지와 글에 적용시켜주었습니다.
    하지만 버튼을 클릭시 자동슬라이드가 충돌하여 문제점이 있었습니다. 아직 해결하지 못했지만 조금 더 공부해서 수정이 필요할 것 같습니다.
