LELABO

RESPONSIVE UI RENEWAL PROJECT

01. overview.

최신 화장품 웹사이트 트렌드보다 랑콤 사이트가 올드하다고 느껴졌습니다.

랑콤의 고유한 분위기를 살리면서 트렌드를 반영하여 잘 표현하고자 했습니다.
또한 디바이스 제약 없이 모두 사용 가능한 반응형 웹사이트를 제작하였습니다.

03. Publishing.

  • 뷰포트 : Pc 1920px / Tablet 1400px / Mobile 768px
  • 반응형으로 제작하여 다양한 기기에서도 적응이 가능
  • flex와 grid를 이용한 웹페이지 레이아웃 제작/제어
  • pc에서의 사용자 접근성을 높이기 위한 마우스 오버 : hover 적용
  • jQuery 플러그인을 이용한 비디오 삽입
  • javascript를 이용한 배너 이미지 슬라이드 구현
  • javascript를 이용한 모바일 DrowDown Menu 구현

04. Review.

  • # 내 첫 프로젝트

    디자인부터 퍼블리싱까지 스스로 모든것을 구현한 첫 프로젝트였습니다.
    만족스러운 결과물이 나오진 못했지만 , 내가 보완해야 할 문제점들을 파악할 수 있어 배운점이 많았습니다.

  • # 다양한 크기와 비율

    퍼블리싱을 할때 mobile, tablet, pc 에서 고려해야할 크기가 다 달라 적용시키기 어려웠습니다.
    gird를 통해 content를 정렬 시킨 후 가변 단위를 이용해 유연한 반응형 웹사이트를 만들었습니다.

  • # 플러그인 사용

    처음에는 비디오 삽입을 iframe 방식으로 적용시켰습니다. 보안 문제로 jQueyr 플러그인을 사용했습니다.
    jQuery 플러그인 이용하여 영상을 깔끔하게 보여주고 컨트롤 바, 공유 등 불필요한 속성들을 제어할 수 있었습니다.
